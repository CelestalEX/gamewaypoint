# GameWaypoint

GameWaypoint is a modern platform designed for gamers to discover, create, and share high-quality gaming guides.

The goal of the platform is to make complex games easier and more enjoyable by giving players access to structured walkthroughs, tutorials, tips, and community-created content.

---

## Features

### Guide Discovery

* Browse game guides and walkthroughs
* Search for tutorials and gameplay tips
* Responsive desktop & mobile experience

### Custom Markdown Editor

* Advanced markdown editor inspired by Obsidian and Notion
* Floating formatting toolbar
* Context actions menu
* Live markdown preview
* Slash commands
* Autosave & draft recovery
* Syntax highlighting
* Image uploads

### Guide Creation System

* Create and publish custom guides
* Rich markdown formatting
* Mobile & desktop preview modes
* Structured content editing workflow
